# Uzman Görüşleri

Bu klasöre başka yapay zeka/kod uzmanlarından aldığınız görüşleri düz metin
veya markdown dosyası olarak koyun. Dosya adı deseni:

```
YYYY-MM-DD-kisa-baslik.md
```

Örnek: `2026-07-16-mimari-elestirisi.md`

Her dosya tek bir görüşü içersin (birden fazla görüşü aynı dosyaya
karıştırmayın — ayrı ayrı değerlendirmek daha sağlıklı olur).

Bir dosya eklediğinizde bana haber verin; her görüşü tek tek okuyup
doğrulanabilir kaynaklarla teyit edip (mümkünse) ana `../PLAN.md` dosyasına
işleyeceğim veya neden reddettiğimi açıklayacağım.
